# DevAI – Static Code Auditor & Optimizer

## Overview

DevAI is an AI-powered static code analysis dashboard built with Streamlit and Google's Gemini model. It helps developers identify bugs, analyze code complexity, and generate optimized code suggestions through a clean and responsive interface.

## Features

* Multi-language code analysis (Python, Java, C++, JavaScript, Go)
* AI-powered bug and issue detection
* Time and space complexity evaluation
* Code optimization recommendations
* Clean, developer-friendly dashboard
* Real-time analysis and feedback

## Tech Stack

* Python
* Streamlit
* Google Gemini 3.5 Flash
* Google GenAI SDK

## Installation

Install the required dependencies:

```bash
pip install streamlit google-genai
```

Configure your API key in:

```toml
.streamlit/secrets.toml

GEMINI_API_KEY = "YOUR_API_KEY"
```

## Run the Application

Start the Streamlit server:

```bash
streamlit run app.py
```

Open the generated local URL in your browser to access the dashboard.

## Workflow

1. Select a programming language.
2. Paste source code into the editor.
3. Run the analysis.
4. Review detected issues, complexity metrics, and optimization suggestions generated by Gemini AI.
