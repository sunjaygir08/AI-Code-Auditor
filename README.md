# DevAI – Static Code Auditor & Optimizer

## Overview

DevAI is an AI-powered static code analysis dashboard built with Streamlit and Google's Gemini models. It helps developers identify bugs, analyze code complexity, and generate optimized code suggestions through a clean, responsive, and theme-adaptive interface that works seamlessly in both light and dark mode.

## Features

* Multi-language code analysis (Python, Java, C++, JavaScript, Go)
* AI-powered bug and issue detection
* Time and space complexity evaluation
* Code optimization recommendations
* Drag-and-drop file uploader for instant code import
* Organized output view with separate **Fixed Code** and **Notes & Metrics** tabs
* One-click download of the optimized code
* Sidebar model selector to switch between Gemini model versions
* Clean, developer-friendly dashboard with full dark/light theme support
* Real-time analysis and feedback

## Tech Stack

* Python
* Streamlit
* Google Gemini (selectable: 3.5 Flash, 3.1 Pro, 3 Flash, 2.5 Pro, 2.5 Flash, 2.5 Flash-Lite)
* Google GenAI SDK

## Installation

Install the required dependencies:

```bash
pip install streamlit google-genai
```

Configure your API key in:

```toml
.streamlit/secrets.toml

GEMINI_API_KEY = "YOUR_API_KEY"
```

## Run the Application

Start the Streamlit server:

```bash
streamlit run app.py
```

Open the generated local URL in your browser to access the dashboard.

## Workflow

1. Select a programming language.
2. Choose a Gemini model from the sidebar (or keep the default).
3. Paste source code into the editor, or drag and drop a code file to auto-fill it.
4. Run the analysis.
5. Review the results across two tabs — **Fixed Code** (with a download option) and **Notes & Metrics** (issue summary, fix applied, time/space complexity, and optimization tip) generated by Gemini.
